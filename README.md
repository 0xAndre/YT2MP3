# YT2MP3

Youtube to MP3 Converter via WebServer with Express <br />
Education, freedom and knowledge - For Educational Purposes Only

### Prerequisites

NodeJS

### Installing

1) Clone Repo

```
git clone https://github.com/x0uter/YT2MP3.git
```

2) Install libs

```
npm install 
```

3) Run

```
npm start 
```

4) Open browser

```
http://localhost:3000
```
